git init -> Inicializa/ CREA un repositorio en local 
git add -> Preparar los archivos que tienen cambios (que no estan en la version actual)
git status -> Ver el estado del repositorio en la rama actual
git commit -m -> Confirma con un mensaje que esos archivos son la version final de esa rama

git branch -> Interactuar con ramas (nos muestra el listado de ramas)
git branch <nombre_nueva_rama> -> Crea una nueva rama
git branch -M <nombre_nuevo> -> Cambiar el nombre de la rama actual
git branch -d  -> Eliminar la rama

git checkout <nombre_rama> -> Nos movemos entre las diferentes ramas
git checkout -b <nombre_rama> -> Crea una nueva rama y nos mueve a ella (quedamos parados en esa rama)

------ GITHUB ----- 
git remote add origin <url> -> Enlaza nuestro repo local con un repositorio remoto
git push -u origin <nombre_rama> -> Envia los cambios a github
git clone <url> -> Clona el repositorio de github y lo descarga donde lo ejecutamos
